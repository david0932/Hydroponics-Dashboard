# **App Name**: HydroView

## Core Features:

- Sensor Data Visualization: Display real-time sensor data (temperature, humidity, pH, water level) in a clear, graphical format. Use dummy data for initial setup.
- Threshold Monitoring: Allow users to set desired ranges for each sensor. Highlight when values fall outside the optimal range.
- Action Logging: Provide a simple interface for users to log actions taken (e.g., nutrient adjustments, water changes) with timestamps.
- AI-Powered Recommendations: Use a generative AI model to provide recommendations for optimizing growth conditions based on current sensor data and logged actions.  The AI model acts as a tool to help the user.
- Historical Data Analysis: Display historical data for each sensor, allowing users to identify trends and patterns over time. Use interactive charts and graphs.

## Style Guidelines:

- Primary color:  Deep Green (#1E3A2B) to evoke nature and growth.
- Secondary color: Light Green (#A1D99B) for accents and highlights.
- Accent color: Teal (#26A69A) for interactive elements and call-to-actions.
- Clean and modern typography for easy readability.
- Use clear and intuitive icons to represent different sensors and actions.
- Dashboard layout with clear sections for real-time data, historical data, and recommendations.
